Backpropagation Math: Manual Attention Gradients

Most deep learning practitioners rely on automatic differentiation (Autograd) in PyTorch or JAX. However, writing custom CUDA kernels or building deep learning frameworks from scratch requires manual derivation of backpropagation gradients.

Let's look at the Scaled Dot-Product Attention:

Attention(Q, K, V) = softmax(Q K^T / sqrt(d_k)) V

Let the intermediate softmax matrix be represented as S. Thus, the attention output is A = S V.

To backpropagate through this module, we start with the incoming gradient with respect to the output, dL/dA. We can compute the gradient with respect to V as:

dL/dV = S^T (dL/dA)

The gradient with respect to the attention scores S requires more care, as it involves the softmax activation:

dL/dS = (dL/dA) V^T

From there, we backpropagate through the softmax function using its Jacobian matrix, and finally compute the gradients with respect to Q and K. Deriving these expressions is a crucial step when writing custom performance kernels in low-level systems.
